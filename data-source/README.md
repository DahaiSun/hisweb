# Data Sources Index

This folder tracks public data sources for the "100 years of financial history" website.

## Existing Source Notes
- Archive.org: `archive.org.md`
- IMF Archives: `imf.archives.md`

## Recommended Public Databases
1. FRED / ALFRED
   - Website: https://fred.stlouisfed.org/
   - API: https://fred.stlouisfed.org/docs/api/fred/fred/
   - Use: macro series and historical revisions.
2. FRASER
   - Website: https://fraser.stlouisfed.org/
   - About: https://fraser.stlouisfed.org/about/index
   - Use: historical financial documents and reports.
3. SEC EDGAR
   - Website: https://www.sec.gov/edgar
   - API Docs: https://www.sec.gov/edgar/sec-api-documentation
   - Use: company disclosures and filing evidence.
4. IMF Data
   - Website: https://data.imf.org/
   - API: https://data.imf.org/en/Resource-Pages/IMF-API
   - Use: international macro-financial indicators.
5. World Bank Indicators
   - Website: https://data.worldbank.org/
   - API Docs: https://datahelpdesk.worldbank.org/knowledgebase/articles/889392-about-the-indicators-api-documentation
   - Use: country-level long-run indicators.
6. BIS Data Portal
   - Website: https://data.bis.org/
   - Use: banking, credit, and financial stability statistics.
7. OECD Data API
   - Website: https://www.oecd.org/en/data/
   - API Guide: https://www.oecd.org/en/data/insights/data-explainers/2024/09/api.html
   - Use: cross-country OECD economic data.
8. ECB Data Portal
   - Website: https://data.ecb.europa.eu/
   - API Help: https://data.ecb.europa.eu/help/api
   - Use: euro area monetary and financial data.
9. U.S. Treasury Fiscal Data
   - Website: https://fiscaldata.treasury.gov/
   - API Docs: https://fiscaldata.treasury.gov/api-documentation/
   - Use: debt and fiscal time series.
10. BEA API
   - Website: https://www.bea.gov/
   - Developer Tools: https://www.bea.gov/resources/developer-tools
   - Use: GDP and national accounts.
11. BLS API
   - Website: https://www.bls.gov/
   - API Features: https://www.bls.gov/bls/api_features.htm
   - Use: CPI, labor market, and inflation context.
12. CFTC Commitments of Traders
   - Website: https://www.cftc.gov/MarketReports/CommitmentsofTraders/index.htm
   - Use: futures positioning and sentiment proxy.
13. FDIC Bank Data API
   - Website: https://www.fdic.gov/
   - API Docs: https://api.fdic.gov/banks/docs
   - Use: U.S. bank-level historical data.
14. Eurostat API
   - Website: https://ec.europa.eu/eurostat/
   - API Intro: https://ec.europa.eu/eurostat/web/user-guides/data-browser/api-data-access/api-introduction
   - Use: EU statistical datasets.
15. GDELT
   - Website: https://www.gdeltproject.org/
   - Use: global event/news detection feed.
16. Library of Congress APIs
   - Website: https://www.loc.gov/
   - API Portal: https://www.loc.gov/apis/
   - Use: historical newspapers, media, and context sources.
